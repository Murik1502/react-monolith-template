## Installation

### Starting project

1. **Clone repository**:

   ```bash
   git clone git@github.com:Murik1502/react-monolit-template.git
	 ```

2. **Install dependencies**:

   ```bash
   npm i
   ```

3. **Start project**:

   ```bash
   npm run start
   ```

### Other Scripts

1. **run analyzer**:

	 ```bash
   npm run analyze
   ```

2. **run ESlint**:
	
	```bash
	npm run lint
	```

3. **run Check**:

	```bash
	npm run check
	```
